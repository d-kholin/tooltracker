CSV Format Details

Column Description Required Example
Name Tool name Required Hammer
Description Tool description Optional Standard claw hammer
Value Tool value in dollars Optional 25.00
Brand Tool brand Optional Stanley
Model Number Tool model number Optional 16-100
Serial Number Tool serial number Optional HM001
Created At Creation timestamp (ISO format) Optional 2024-01-15T10:00:00

Important: If you don't specify a creation date, the current date and time will be used. The Created At field should be in ISO format (YYYY-MM-DDTHH:MM:SS) or left empty.
